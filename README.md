# Quiz Server

[![Build Status](https://travis-ci.org/michaelrausch/LAN-Manager.svg?branch=master)](https://travis-ci.org/michaelrausch/LAN-Manager)

## Installation
1. Install NodeJS
2. Edit config/settings.json
3. Add your questions to config/settings.json
4. npm install
5. node bin/www
